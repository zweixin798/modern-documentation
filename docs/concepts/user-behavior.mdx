# User Behavior

## 1. 用户第一次使用 AI

大多数用户会做三件事：

- 测试能力
- 问简单问题
- 看 AI 是否可靠

---

## 2. 用户流失原因

常见原因：

- 第一次结果不好
- 不会写 prompt
- 结果不稳定

---

## 3. 留存关键点

First Success（第一次成功体验）

如果用户第一次得到满意结果
留存概率会上升

---

## 4. 产品需要做的事情

降低门槛：

- 提供示例 prompt
- 自动补全

提高成功率：

- 限制输入方式
- 提供模板

---

## 5. 用户行为路径

尝试 → 成功 → 复用 → 依赖
