# Token Economics

## 1. Core Formula

AI 产品成本基本结构：

Cost ∝ Token 数量

---

## 2. Token 来源

主要来自两个部分：

- Input Token（用户输入）
- Output Token（模型输出）

---

## 3. 成本 vs 体验

增加 token 会发生什么：

- 上下文变长 → 质量提升
- 输出更长 → 成本增加
- 历史对话更多 → 连贯性更好

---

## 4. 产品优化策略

控制成本：

- 限制上下文长度
- 截断历史记录

提升效果：

- 优化 prompt
- 结构化输入

---

## 5. 产品矛盾

更好的 AI 体验
通常意味着

更高的推理成本
