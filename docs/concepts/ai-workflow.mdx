# AI Workflow

## 1. System Overview

AI 产品的核心不是模型，而是一个 pipeline：

User Input → Prompt → Model → Output → User Feedback

---

## 2. Key Layers

### Input Layer（用户输入）

- 用户目标不明确
- 输入质量波动大

---

### Prompt Layer（控制层）

- 决定输出结构
- 是产品设计核心

---

### Model Layer（生成层）

- 概率生成
- 成本由 token 决定

---

### Output Layer（结果层）

- 用户感知价值
- 决定留存

---

## 3. Failure Points

为什么 AI 产品不稳定：

- prompt 不稳定
- 输入不一致
- 模型随机性

---

## 4. Product Implication

产品优化方向：

- 控制输入
- 设计 prompt
- 降低随机性
